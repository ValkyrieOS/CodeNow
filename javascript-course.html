Curso de JavaScript

Bienvenido al curso de JavaScript. Este curso está diseñado para principiantes y te llevará desde los conceptos básicos hasta temas más avanzados.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite crear contenido interactivo en páginas web. Es uno de los lenguajes más populares y versátiles en el desarrollo web.

Tu primer programa en JavaScript

Empecemos con algo simple: mostrar un mensaje en la consola.

console.log("¡Hola, mundo!");

Para probar este código:

  1. Abre las herramientas de desarrollo de tu navegador (F12)
  2. Ve a la pestaña "Consola"
  3. Escribe el código y presiona Enter

Variables y tipos de datos

Las variables son contenedores para almacenar datos:

// Números let edad = 25; // Texto (strings) let nombre = "María"; // Booleanos let estaEstudiando = true; // Arrays let colores = ["rojo", "verde", "azul"]; // Objetos let persona = { nombre: "Juan", edad: 30 };

Ejemplo práctico

Veamos un ejemplo simple de cómo JavaScript puede hacer una página interactiva:

function saludar() { let nombre = prompt("¿Cómo te llamas?"); alert("¡Hola, " + nombre + "!"); }
